# Avito-parser-Bot
Парсер Авито через телеграмм бота. Для пирсинга будут использованы библиотеки "requests и BeautifulSoup". Бот будет работать через библиотеку "aiogram"


Пример работы:
user: Начать
bot: Что будем искать?
user: Геймпад Xbox
bot: В каком городе?
user: Москва 
bot: Выполняю поиск...
bot: Вот что удалось найти:
     ( *ссылки на "геймпады Xbox" в Москве* )

Также возможно расширение функционала в виде добавление дополнительных
фильтров поиска, что есть в Авито ( радиус поиска, ценовой диапазон)
